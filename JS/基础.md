# js基础

记录容易混淆的内容

## 变量

变量名称可以由字母、数字、下划线、$组成，但是不能以数字开头

```js
var a
var thisIsAVariable
var _type_ty_ty
var type2type
var 2thx // 报错，变量不能以数字开头
```

## 操作符

后置的++操作会先返回该值，然后再增加1

```js
var a = 123
var b = a++

console.log(a)  // 124
console.log(b)  // 123
```

前置的++操作会先将值增1，然后再返回

```js
var a = 123
var b = ++a

console.log(a)  // 124
console.log(b)  // 124
```
